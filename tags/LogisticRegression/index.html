In this project, I built machine learning model that best identifies potential donors for CharityML(a fictitious charity organization) with data collected for the U.S. census. To find the best approach, I performed EDA, feature engineering, and building training and predicting pipeline to evaluate and optimize the performance between different machine learning models.

The model I have used:

  • SGD Classifier
  • AdaBoost
  • Logistic Regression
